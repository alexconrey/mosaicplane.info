Quick Start - Key Endpoints:

GET /api/v1/aircraft/ - List all aircraft with filtering options
GET /api/v1/manufacturers/ - List aircraft manufacturers
GET /api/v1/aircraft/{id}/ - Get detailed aircraft specifications
POST /api/v1/corrections/ - Submit data corrections

About MOSAIC: The MOSAIC regulations represent a revolutionary change in general aviation, expanding sport pilot privileges to approximately 70% of the existing GA fleet. This API helps pilots understand which aircraft they can legally operate under different certificate levels.
